Vì bạn đang dùng **Desktop miễn phí**, nút **Download codebase** trong Code Editor thường sẽ không hiện. Bạn có 2 cách phù hợp để lấy full source code:

1. **Cách khuyến nghị: kết nối GitHub**
   - Trong khung chat của Lovable, bấm nút **+** ở góc dưới.
   - Chọn **GitHub** → **Connect project**.
   - Đăng nhập/ủy quyền GitHub.
   - Tạo repository cho project.
   - Sau khi sync xong, mở repository trên GitHub.
   - Bấm **Code** → **Download ZIP** để tải full source code.

2. **Nếu muốn tải trực tiếp trong Lovable**
   - Cần dùng workspace trả phí.
   - Khi có quyền tải, vào **Code Editor**.
   - Nhìn xuống cuối cây thư mục bên trái sẽ thấy nút **Download codebase**.

3. **Lưu ý quan trọng**
   - Source code chỉ gồm mã nguồn frontend/app.
   - Dữ liệu backend/database không nằm trong ZIP source code.
   - Nếu cần xuất dữ liệu backend, vào **Cloud** → **Advanced settings** → **Export data**.

Tóm lại: với tài khoản **Desktop miễn phí**, bạn nên dùng **GitHub sync**, sau đó tải ZIP từ GitHub.
